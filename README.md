# Software-Defined Radio (SDR) & Digital Communications

This repository serves as my portfolio for Radio Frequency (RF) engineering and Digital Signal Processing (DSP). It contains real-time communication system simulations and hardware implementations bridged via GNU Radio and SDR platforms (e.g., PlutoSDR/RTL-SDR).

## 📡 Project Overview & Implementations

My focus in these projects is to observe the physical behavior of mathematical formulas and theoretical communication models. Key implementations include:

* **Digital Modulation Schemes:** Implementation and analysis of multi-carrier systems and various digital modulation techniques (e.g., QPSK, QAM).
* **Channel Estimation & Synchronization:** Observing the real-world effects of channel conditions, noise, and fading on signal integrity.
* **Spectrum Analysis:** Real-time monitoring of the electromagnetic spectrum and bandwidth utilization.
* **Constellation Observation:** Hardware-in-the-loop testing to analyze constellation diagrams and system performance (EVM, BER estimations).

## 🛠️ Technology Stack
* **Software:** GNU Radio Companion (GRC), DSP Algorithms
* **Hardware Interfacing:** SDR (Software-Defined Radio) architecture
* **Core Concepts:** RF Engineering, Digital Communications, Signal Processing, Telemetry
